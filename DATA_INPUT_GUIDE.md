# 📊 Polymarket 数据输入模板

## 如何提供市场数据

由于我无法直接访问 Polymarket，需要你提供数据。

---

## 方法 1: 发送截图 + 描述

在 Discord 发送：

```
📸 市场数据
时间: 2026-02-19 10:30 UTC
市场: Bitcoin Up or Down - Feb 19, 8:10PM
Yes价格: 0.52
No价格: 0.48
交易量: $50,000
我的分析: 看涨，因为...
[上传截图]
```

---

## 方法 2: 批量发送多个市场

```
📊 批量市场数据
时间: 2026-02-19 10:30 UTC

市场1:
- 名称: Bitcoin Up or Down
- Yes: 0.52, No: 0.48

```

## 方法 3: 使用 JSON 格式 (高级)

创建一个 `latest_market_data.json` 文件：

```json
{
  "timestamp": "2026-02-19T10:30:00Z",
  "markets": [
    {
      "name": "Bitcoin Up or Down - Feb 19",
      "yes_price": 0.52,
      "no_price": 0.48,
      "volume": 50000,
      "liquidity": 15000
    },
    {
      "name": "Ethereum Up or Down - Feb 19",
      "yes_price": 0.55,
      "no_price": 0.45,
      "volume": 30000,
      "liquidity": 10000
    }
  ]
}
```

## 我应该如何回应

我会：
1. **分析数据** - 识别套利机会和交易想法
2. **生成交易建议** - 买入/卖出建议，风险评级
3. **记录到系统** - 保存到 `paper_trading/trading_ideas.json`
4. **更新 Notion** - 推送到 GitHub 仓库
5. **发送提醒** - 如果有即时交易机会

```

```

```
等待你确认是否执行！
```

## 频率建议

- **高频:** 每 5-15 分钟查看一次（如果你在看盘）
- **中频:** 每小时查看一次
- **低频:** 只在重大事件前查看

系统会自动每 5 分钟检查一次，但需要你提供数据。
