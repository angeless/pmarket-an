# 💰 预测市场赚钱策略手册

**目标:** 小本金 ($100-1000) 在 Polymarket/Kalshi 赚钱
**策略:** 低风险套利 + 情绪对冲
**预期月收益:** 10-25%

---

## 一、跨平台对冲套利 (低风险)

### 原理
同一事件在不同预测平台的价格差异。

### 操作
1. **找事件** - 同时在 Polymarket 和 Kalshi 上架
2. **比价格** - 记录两边的 Yes/No 价格
3. **算价差** - 如果总和 < 1.0，存在套利空间
4. **操作** - 在低价平台买，高价平台卖

### 示例
```
事件: "美联储3月降息"
Polymarket: Yes 0.60 | No 0.42 (总和 1.02)
Kalshi:     Yes 0.58 | No 0.40 (总和 0.98)

套利: 在 Kalshi 买 Yes (0.58), 在 Polymarket 卖 Yes (0.60)
利润: 0.02 / 0.58 = 3.4% (无风险)
```

### 风险
- 平台手续费 (通常 2%)
- 结算时间差
- 流动性不足

## 二、社交媒体情绪套利 (中风险)

### 原理
社交媒体情绪与市场价格偏离。

### 操作
1. **选事件** - 高关注度 (大选、宏观政策)
2. **采情绪** - 小红书/X/微博关键词分析
3. **算偏离** - 情绪偏向 vs 市场价格
4. **下注** - 情绪-价格偏离 > 10% 时入场

### 示例
```
事件: "特朗普2024胜选"
小红书情绪: 70% 认为会赢
Polymarket价格: Yes 0.52 (52% 概率)
偏离: 70% - 52% = 18%

操作: 买 Yes (市场低估)
预期: 价格回归至 0.65-0.70
收益: 25-35%
```

### 风险
- 情绪判断错误
- 价格长期偏离
- 黑天鹅事件

## 三、高流动性做市 (高风险)

### 原理
在流动性好的市场双边挂单，赚价差。

### 操作
1. **选市场** - 交易量 > $1M，波动适中
2. **双边挂** - Yes 0.48 买 / 0.52 卖
3. **吃单费** - 有人成交即赚手续费返还
4. **高频操作** - 每日多次调整

### 示例
```
市场: "BTC 月底价格"
挂单: Buy Yes @ 0.48, Sell Yes @ 0.52
价差: 4%

如果有人两边都吃，利润 4%
如果只吃一边，等价格回归
```

### 风险
- 单边暴露
- 价格剧烈波动
- 需要盯盘

## 四、事件驱动策略 (中风险)

### 原理
利用信息不对称，在新闻/数据发布前下注。

### 操作
1. **盯日历** - FOMC、CPI、就业数据
2. **预判** - 基于历史数据预测结果
3. **速下注** - 数据发布前 5-10 分钟
4. **快离场** - 发布后 30 分钟内平仓

### 示例
```
事件: "2月CPI > 3%"
历史规律: 1月CPI通常高于预期
操作: 在数据发布前 10 分钟买 Yes
预期: 发布后价格跳涨
```

### 风险
- 预测错误
- 滑点大
- 无法及时离场
