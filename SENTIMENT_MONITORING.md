## 三、情绪指标计算

### 情绪-价格偏离公式
```
偏离度 = 社交媒体情绪% - 市场价格%

示例:
小红书认为美联储降息概率: 70%
Polymarket市场价格: 52%
偏离度 = 70% - 52% = 18%

判断:
偏离度 > 10% → 市场情绪过热，考虑反向操作
偏离度 < -10% → 市场情绪过冷，考虑正向操作
偏离度在 ±10% → 情绪与价格一致，观望
```

### 情绪评分标准
| 情绪倾向 | 评分 | 说明 |
|----------|------|------|
| 极度看涨 | +5 | >80% 认为事件发生 |
| 看涨 | +3 | 60-80% 认为事件发生 |
| 中性 | 0 | 40-60% 摇摆 |
| 看跌 | -3 | 20-40% 认为事件发生 |
| 极度看跌 | -5 | <20% 认为事件发生 |
