# Kalshi 研究报告

## 基本信息

**平台:** Kalshi  
**网址:** https://kalshi.com  
**类型:** 美国合规预测市场  
**监管:** CFTC 批准（美国商品期货交易委员会）

## 与 Polymarket 对比

| 特性 | Kalshi | Polymarket |
|------|--------|------------|
| 合规性 | ✅ 美国合规 | ⚠️ 灰色地带 |
| KYC | 需要 | 不需要 |
| 费用 | 低 | 中等 |
| 市场类型 | 经济/天气/政治 | 更广泛 |
| 提款 | 容易 | 需要加密货币 |
| API | 可能有 | 有限 |

## 优势

1. **合规安全** - 受 CFTC 监管，资金安全
2. **美元结算** - 无需处理加密货币
3. **税务清晰** - 1099 表格自动报告
4. **提款方便** - ACH 直接到银行账户

## 劣势

1. **市场较少** - 监管限制，市场类型有限
2. **KYC 严格** - 需要身份验证
3. **地理限制** - 仅限美国用户
4. **流动性较低** - 用户基数较小

## 套利机会

### 跨平台套利 (Kalshi ↔ Polymarket)
- 同一事件（如选举）在两个平台定价差异
- Kalshi 通常流动性较低，定价效率不足
- 预期套利空间: 2-5%

### 策略建议

**小本金 ($500):**
1. 优先选择 Kalshi 的宏观经济事件
2. 对比 Polymarket 价格差异
3. 在 Kalshi 下注（流动性低但波动大）
4. 在 Polymarket 对冲（流动性高）

**风险提示:**
- Kalshi 提款可能需要 3-5 天
- 汇率风险（美元计价）
- 流动性风险（大单难以成交）

## 结论

Kalshi 是 Polymarket 的良好补充，特别是对于：
- 不想处理加密货币的用户
- 需要合规税务报告的用户
- 专注美国经济/政治事件的用户

**建议:** 两个平台结合使用，Kalshi 作为 Polymarket 的对冲/套利工具。
